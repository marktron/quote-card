# Extension Icons

This directory should contain the following icon files:

- `icon-16.png` - 16x16px (toolbar icon in Safari)
- `icon-32.png` - 32x32px (retina toolbar icon)
- `icon-48.png` - 48x48px (extension manager, notifications)
- `icon-128.png` - 128x128px (Mac App Store, extension details)

## Design Guidelines

The icon should represent quote marks or text selection in a clean, modern style that fits with macOS design language.
